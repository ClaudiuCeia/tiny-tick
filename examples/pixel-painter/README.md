# Pixel Painter

Paint pixels on a grid canvas. Every painted pixel is autosaved to `localStorage`.

## Run

```bash
bun run example:pixel-painter
```

Then open `http://localhost:5176/`.

## Controls

- Hold left mouse button and drag on the canvas to paint.
- Use the color input to pick the paint color.
- Click **Clear** to erase the board and remove saved data from `localStorage`.
